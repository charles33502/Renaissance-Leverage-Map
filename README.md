# Renaissance Leverage Map (RLM)
# 文藝復興槓桿地圖
[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19915464.svg)](https://doi.org/10.5281/zenodo.19915464)

**Renaissance Leverage Map (RLM)**, also known in Chinese as **文藝復興槓桿地圖** and informally as **復興表**, is a four-column framework for identifying AI-era leverage points: where AI lowers execution costs and makes classical capabilities newly scalable.

**Renaissance Leverage Map（RLM，文藝復興槓桿地圖）**，中文可簡稱為**復興表**，是一個四欄分析框架，用來辨識 AI 時代中，哪些古典能力因為執行成本下降，而重新變成高槓桿能力。

---

## Core Idea / 核心概念

AI does not necessarily replace fundamentals.  
In many cases, AI lowers the execution cost required to revive and scale classical capabilities that were already valuable but previously too expensive, slow, or difficult to apply consistently.

AI 不一定取代基本功。  
更多時候，AI 降低了執行成本，使得許多原本重要、但過去太昂貴、太慢、太麻煩、難以持續實作的古典能力，重新變得可規模化、可日常化、可高頻使用。

## Origin and Timestamp / 起源與時間戳

This repository documents the initial formulation of **Renaissance Leverage Map**, a personal framework for analyzing AI-era capability investment.

The framework was created to answer the question:

> When AI lowers the execution cost of a task, which classical capability becomes newly valuable again?

本 repository 用於記錄 **Renaissance Leverage Map（文藝復興槓桿地圖）** 的初始版本。

這個框架試圖回答：

> 當 AI 降低某件事的執行成本時，哪些原本就有價值的古典能力會重新變成高槓桿能力？

This repository is maintained as a timestamped public record of the framework's development.

本 repository 主要目的之一，是留下此框架發展過程的公開時間戳與版本紀錄。
